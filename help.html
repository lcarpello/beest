How to navigate the input options.

The first input box is used to enter: The first value for addition and subtraction, salary, amount of money effected by the interest, and miles

The second input box is used to enter: The second value for addition and subtraction, and amount of time that the interest takes place

The third input box is used to enter: The interest rate in percent per year


EXAMPLES:

sum: first + second = result, 5 + 5 = 10


minus: first - second = result, 10 - 5 = 5


VAT: first x 1.2 = result, 100 x 1.2 = 120


interest: first x second x (third ÷ 100) + first = result, 1000 x 5 x 0.05 + 1000 = 1250


salary: first ÷ 365.25 = result, 36525 ÷ 365.25 = 100


from miles: first x 1.609344 = result, 1 x 1.609344 = 1.609344
